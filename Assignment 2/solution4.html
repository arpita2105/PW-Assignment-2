LIST OF HTML TAGS

  1. <!DOCTYPE> : Specifies the document type and version of HTML.
  2. <html> : The root element that wraps the entire HTML document.
  3. <head> : Contains metadata about the document, such as the title, character set, and linked resources.
  4. <title> : Sets the title of the web page, displayed in the browser's title bar or tab.
  5. <body> : Contains the visible content of the web page, including text, images, and other elements.
  6. <h1> to <h6> : Headings, with <h1> being the highest and <h6> the lowest, used to structure content.
  7. <p> : Represents a paragraph of text.
  8. <a> : Creates hyperlinks to other web pages or resources.
  9. <img> : Embeds images on the page.
  10. <div> : A container used for grouping and styling elements.
  11. <table> : Defines a table for tabular data.
  12. <form> : Used for creating input forms.
  13. <hr> : Inserts a horizontal line or thematic break.
  14. <br> : Inserts a line break within text.
